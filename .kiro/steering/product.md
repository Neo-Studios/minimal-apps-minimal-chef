# Product Overview

Zest is a modern multi-platform cooking application with native implementations for Web, Android, and iOS. Version 2.0 represents a complete native rewrite for better performance and platform-specific features.

## Core Purpose
Recipe management, meal planning, and smart cooking assistance with on-device AI capabilities across 8 AI providers (Google, Samsung, Xiaomi, OPPO, OnePlus, Huawei, Qualcomm, Apple).

## Key Features
- Recipe management (browse, create, import from URLs)
- Calendar-based meal planning
- Smart shopping lists with Instacart integration
- Recipe cookbooks and collections
- On-device AI for recipe suggestions and photo analysis
- Nutrition tracking with calorie counter and macros
- Voice-guided cooking with hands-free mode
- Multiple cooking timers with notifications
- Unit conversion and recipe scaling
- Full offline mode with cloud sync

## Accessibility Focus
Strong emphasis on accessibility with dyslexia-friendly fonts, haptic feedback, voice input, screen reader support, high contrast mode, and color blind modes (Protanopia, Deuteranopia, Tritanopia).

## Internationalization
Supports 21 languages including English (US, UK, AU), French, Spanish, Italian, German, Dutch, Swedish, Portuguese, Japanese, Chinese, Korean, Russian, Polish, Turkish, Arabic, Hindi, Vietnamese, Thai, and Indonesian.

## Backend
Firebase-based with Authentication (Google Sign-In), Cloud Firestore, and Firebase Storage. All platforms support offline-first architecture with local persistence.
