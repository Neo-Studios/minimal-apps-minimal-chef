# Shared Icons

Place app icons and assets here for export to each platform.

## Required Icons

- App icon (1024x1024 base)
- Favicon (web)
- Android adaptive icons
- iOS app icons (various sizes)

## Export Targets

- **Web**: public/favicon.ico, public/icon-*.png
- **Android**: app/src/main/res/mipmap-*/ic_launcher.png
- **iOS**: Assets.xcassets/AppIcon.appiconset/
