The Zest iOS app is built with SwiftUI.

## Tech Stack

*   **Language:** Swift
*   **UI:** SwiftUI with a custom "Liquid Glass" design language
*   **Architecture:** MVVM
*   **Frameworks:** Combine

## Getting Started

1.  Open the `ios/Zest.xcworkspace` file in Xcode.
2.  Set up Firebase:
    *   Download the `GoogleService-Info.plist` file from your Firebase project's settings.
    *   Place the `GoogleService-Info.plist` file in the `ios/Zest` directory.
3.  Build and run the app on a simulator or a physical device.

## Testing

The iOS app includes unit tests and UI tests. You can run these tests from Xcode.
