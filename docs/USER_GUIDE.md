# Zest User Guide

## Getting Started

### Installation

#### Web
Visit [zest.app](https://zest.app) in your browser. For the best experience, install as a Progressive Web App:
1. Click the install icon in your browser's address bar
2. Follow the prompts to add Zest to your home screen

#### Android
1. Download from Google Play Store
2. Open the app and sign in with Google

#### iOS
1. Download from the App Store
2. Open the app and sign in with Google

### First Time Setup

1. **Sign In**: Use your Google account to sign in
2. **Grant Permissions**: Allow notifications for timer alerts and meal reminders
3. **Explore**: Browse the recipe library or add your first recipe

## Features

### Recipe Management

#### Browsing Recipes
- Navigate to the **Recipes** tab
- Use the search bar to find specific recipes
- Filter by cuisine type, meal type, or dietary restrictions
- Tap any recipe to view full details

#### Adding a Recipe
1. Tap the **+** button on the Recipes screen
2. Fill in recipe details:
   - Name and description
   - Prep and cook time
   - Number of servings
   - Ingredients with amounts and units
   - Step-by-step instructions
3. Optionally add a photo
4. Tap **Save**

#### Importing from URL
1. Tap the **Import** button
2. Paste a recipe URL
3. Review the imported data
4. Edit if needed and save

#### Scaling Recipes
- Open any recipe
- Use the **+** and **-** buttons to adjust servings
- Ingredient amounts update automatically
- Tap **Reset** to return to original servings

#### Organizing with Cookbooks
- Create custom cookbooks to organize recipes
- Add recipes to multiple cookbooks
- Share cookbooks with family and friends

### Meal Planning

#### Planning Your Week
1. Go to the **Meal Plan** tab
2. Select a date on the calendar
3. Tap **Add Meal** to choose a recipe
4. Select meal type (breakfast, lunch, dinner, snack)
5. Repeat for other days

#### Viewing Your Plan
- Calendar view shows all planned meals
- Tap any day to see meal details
- Edit or remove meals as needed

### Shopping Lists

#### Creating a Shopping List
1. Navigate to the **Shopping** tab
2. Manually add items by typing and tapping **Add**
3. Or generate from meal plan:
   - Tap **Generate from Meal Plan**
   - Select date range
   - Review and edit the generated list

#### Managing Your List
- Check off items as you shop
- Swipe to delete items
- Items are organized by category
- Unchecked items appear at the top

#### Instacart Integration
- Tap **Send to Instacart** to open your list in Instacart
- If the app is installed, it opens directly
- Otherwise, opens in your browser

### Cooking Features

#### Multiple Timers
1. While viewing a recipe, tap **Add Timer**
2. Name your timer (e.g., "Pasta", "Sauce")
3. Set duration in minutes
4. Start, pause, or reset as needed
5. Receive notifications when timers complete

#### Voice Guidance
- Enable hands-free mode while cooking
- Voice commands read instructions aloud
- Navigate steps with voice or gestures

### Nutrition Tracking

#### Viewing Nutrition Info
- Each recipe shows calorie and macro information
- Tap **Nutrition** to see detailed breakdown
- Track daily intake in the **Nutrition Dashboard**

#### Setting Goals
1. Go to **Settings** > **Nutrition Goals**
2. Set daily targets for:
   - Calories
   - Protein
   - Carbohydrates
   - Fat
3. View progress in the dashboard

### Settings & Preferences

#### Appearance
- **Theme**: Choose Light, Dark, or System
- **Font Size**: Adjust for readability
- **Dyslexia-Friendly Font**: Enable OpenDyslexic font
- **Color Blind Mode**: Choose from Protanopia, Deuteranopia, or Tritanopia

#### Accessibility
- **Screen Reader**: Optimized for VoiceOver/TalkBack
- **Haptic Feedback**: Enable tactile responses
- **High Contrast**: Improve visibility
- **Voice Input**: Dictate recipes and notes

#### Notifications
- **Meal Reminders**: Get notified before meal times
- **Timer Alerts**: Receive cooking timer notifications
- **Recipe Suggestions**: Daily recipe recommendations

#### Language
Choose from 21 supported languages including:
- English (US, UK, AU)
- Spanish, French, German, Italian
- Japanese, Chinese, Korean
- And more...

## Tips & Tricks

### Quick Actions
- **Long press** a recipe to add to cookbook
- **Swipe** shopping list items to delete
- **Double tap** to scale recipe to common serving sizes

### Offline Mode
- All data syncs automatically when online
- Access recipes, meal plans, and shopping lists offline
- Changes sync when connection is restored

### Sharing
- Share recipes via text, email, or social media
- Export recipes as PDF
- Collaborate on meal plans with family

### Keyboard Shortcuts (Web)
- `Ctrl/Cmd + K`: Quick search
- `Ctrl/Cmd + N`: New recipe
- `Ctrl/Cmd + P`: Print recipe
- `Ctrl/Cmd + S`: Save changes

## Troubleshooting

### Sync Issues
- Check internet connection
- Sign out and sign back in
- Clear app cache in Settings

### Missing Recipes
- Ensure you're signed in with the correct account
- Check if recipes are in a specific cookbook
- Try refreshing the recipe list

### Timer Not Working
- Grant notification permissions
- Check device sound settings
- Ensure app is not in battery saver mode

### Import Failures
- Verify the URL is correct
- Some websites block scraping
- Try copying recipe manually
