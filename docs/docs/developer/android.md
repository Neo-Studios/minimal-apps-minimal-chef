The Zest Android app is built with Jetpack Compose.

## Tech Stack

*   **Language:** Kotlin
*   **UI:** Jetpack Compose with Material 3 Expressive
*   **Architecture:** MVVM + Clean Architecture
*   **Dependency Injection:** Hilt

## Getting Started

1.  Open the `android` directory in Android Studio.
2.  Set up Firebase:
    *   Download the `google-services.json` file from your Firebase project's settings.
    *   Place the `google-services.json` file in the `android/app` directory.
3.  Build and run the app on an emulator or a physical device.

## Testing

The Android app includes unit tests and instrumentation tests. You can run these tests from Android Studio.
