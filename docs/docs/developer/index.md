Welcome to the developer documentation for Zest. This guide provides information about the project's architecture, technologies, and development conventions.

## Overview

Zest is a multi-platform cooking application with native implementations for Web, Android, and iOS. It features recipe management, meal planning, smart shopping lists, and AI-powered assistance.

The project is structured as a monorepo with three separate frontends (`web`, `android`, `ios`) that share a common backend on Firebase.

## Getting Started

To get started with the Zest project, you'll need to have the following tools installed:

*   Node.js
*   npm
*   Android Studio
*   Xcode
*   Firebase CLI

Once you have these tools installed, you can clone the repository and follow the setup instructions for each platform.
