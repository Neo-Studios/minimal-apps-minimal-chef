# Product Overview

## Project Identity
**Name:** Zest  
**Version:** 0.1.0  
**Type:** Native multi-platform cooking application  
**Owner:** Neo Studios

## Purpose and Value Proposition
Zest is a modern cooking application designed to revolutionize the cooking experience through native implementations across Web, Android, and iOS platforms. The v2.0 rewrite delivers 3x faster performance than cross-platform frameworks while providing platform-specific features and design systems.

## Key Features and Capabilities

### Core Recipe Management
- Browse, create, and import recipes from URLs
- Calendar-based meal planning and tracking
- Smart shopping lists with Instacart integration
- Custom recipe cookbooks and collections
- Advanced search with filters (cuisine, dietary restrictions, time, rating)

### Smart AI-Powered Features
- **On-Device AI Support** - 8 AI providers (Google AICore, Samsung Galaxy AI, Xiaomi HyperOS AI, OPPO AndesGPT, OnePlus AI, Huawei Pangu AI, Qualcomm AI Engine, Apple Intelligence)
- Recipe suggestions and recommendations
- Photo analysis for ingredient recognition
- Automatic instruction generation
- Voice-guided cooking with hands-free commands

### Health and Nutrition
- Calorie counter and nutrition tracking
- Macros and nutrition dashboard
- HealthKit integration (iOS)
- Dietary restriction support

### Cooking Assistance
- Multiple simultaneous cooking timers with notifications
- Built-in metric/imperial unit conversion
- Automatic recipe scaling for serving sizes
- Voice input for dictating recipes
- Offline mode with full functionality

### Accessibility Features
- OpenDyslexic font option for dyslexia-friendly reading
- Haptic feedback for tactile responses
- Voice input and screen reader support
- High contrast mode for better visibility
- Color blind modes (Protanopia, Deuteranopia, Tritanopia)

### Multi-Platform Sync
- Firebase real-time cloud synchronization
- Progressive Web App with offline support
- Material 3 Expressive design (Android)
- Liquid Glass design language (iOS)

## Target Users and Use Cases

### Primary Users
- Home cooks seeking organized recipe management
- Health-conscious individuals tracking nutrition
- Busy professionals planning meals efficiently
- Users with accessibility needs (dyslexia, visual impairments, mobility limitations)
- Multi-device users requiring cloud sync

### Use Cases
1. **Daily Cooking** - Follow recipes with voice guidance and timers
2. **Meal Planning** - Plan weekly meals and generate shopping lists
3. **Recipe Discovery** - Find recipes matching dietary needs and available ingredients
4. **Nutrition Tracking** - Monitor calorie intake and macros
5. **Recipe Organization** - Build personal cookbook collections
6. **Hands-Free Cooking** - Use voice commands while cooking
7. **Offline Cooking** - Access recipes without internet connection
