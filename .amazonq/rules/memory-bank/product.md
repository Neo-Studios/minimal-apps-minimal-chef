# Product Overview - Minimal Chef

## Purpose
Minimal Chef is a minimalist cooking application that helps users discover recipes, plan meals, and manage shopping lists with a clean, intuitive interface. The app focuses on simplicity while providing powerful features for home cooks.

## Value Proposition
- **Simplified Cooking**: Streamlined recipe management without overwhelming complexity
- **Multi-Platform**: Works seamlessly across Android, iOS, Web, and Desktop platforms
- **Cloud Sync**: Firebase-powered synchronization keeps data accessible everywhere
- **Modern Design**: Material 3 design with light/dark themes and responsive layouts

## Key Features

### Recipe Management
- Browse recipes from multiple cuisines
- Add custom recipes manually with images
- Import recipes from URLs using web scraping
- Upload recipe photos from camera or gallery
- View detailed recipe information with ingredients and instructions
- Cloud storage for recipe images via Firebase Storage

### Meal Planning
- Calendar-based meal planning interface using table_calendar
- Log and track daily meals visually
- Organize meals by date with intuitive navigation
- Real-time synchronization across devices

### Shopping Lists
- Create and manage ingredient shopping lists
- Mark items as purchased with checkbox interface
- Delete items from list
- Instacart integration for grocery ordering

### Authentication & Sync
- Google Sign-In authentication via Firebase Auth
- Cloud Firestore for real-time data persistence
- Automatic synchronization across all user devices
- Secure user data management

### User Experience
- Material 3 design system with custom color schemes
- Light and dark theme support
- Responsive layouts for mobile, tablet, and desktop
- Custom RobotoFlex variable typography
- Edge-to-edge display on Android
- Springy button animations for tactile feedback

## Target Users
- Home cooks looking for simple meal organization
- Users who want to digitize their recipe collections
- People planning weekly meals and shopping trips
- Multi-device users needing cloud synchronization

## Use Cases
1. **Recipe Discovery**: Browse and save recipes from various cuisines
2. **Meal Planning**: Plan weekly meals using the calendar interface
3. **Shopping Preparation**: Generate shopping lists from planned meals
4. **Recipe Import**: Quickly add recipes from cooking websites
5. **Cross-Device Access**: Access recipes and plans from phone, tablet, or computer

## Current Version
**v0.1.0** - First public release with core recipe management, meal planning, and shopping list features
